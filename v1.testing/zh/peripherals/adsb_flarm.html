# ADS-B/FLARM Receivers

PX4 supports simple air traffic avoidance in missions using ADS-B or FLARM transponders that have a MAVLink interface (that emits the ADSB_VEHICLE message)).

For more information on supported peripherals see Air Traffic Avoidance: ADSB/FLARM.
