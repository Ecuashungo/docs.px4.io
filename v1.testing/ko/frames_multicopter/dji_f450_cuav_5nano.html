# DJI FlameWheel 450 + CUAV V5 nano 구성

이 주제는 키트의 조립과 QGroundControl을 이용하는 PX4 설정에 대한 지침을 제공합니다.

핵심정보

  • 프레임: DJI F450
  • 비행 조절장치: CUAV V5 nano
  • 조립 시간 (예상): 90 분 (프레임 조립에 45분, 오토파일럿 설치와 설정에 45분)

# Bill of materials

이 부품들이 조립에 필요합니다:

  • 비행컨트롤러: CUAV V5 nano:
    • GPS: CUAV NEO V2 GPS
    • 전원 모듈
  • 프레임: DJI F450
  • 프로펠러: DJI Phantom Built-in Nut Upgrade Propellers 9.4x5
  • 배터리: Turnigy High Capacity 5200mAh 3S 12C Lipo Pack w/XT60
  • 텔레메트리: Holibro Transceiver Telemetry Radio V3
  • RC 수신기: FrSky D4R-II 2.4G 4CH ACCST Telemetry Receiver
  • 모터: DJI E305 2312E Motor (800kv,CW)
  • ESC: Hobbywing XRotor 20A APAC Brushless ESC 3-4S For RC Multicopters

추가적으로 우리는 FrSky Taranis 조종기 사용합니다. 부수적으로 케이블 타이와, 양면테이프, 땜 납도 필요합니다.

아래의 이미지는 프레임과 전자 부품을 보여줍니다.

# 필요한 공구

아래 공구들이 조립에 필요합니다.

  • 2.0mm 육각 스크류드라이버
  • 3mm Phillips 스크류드라이버
  • Wire 커터
  • 정밀 트위저
  • 납땜기

# 조립

예상 조립 시간은 약 90 분입니다 (프레임 조립에 약 45 분, 오토파일럿 설치와 구성에 약 45 분).

  1. 제공된 나사를 이용하여 밑판에 팔 4개를 결합합니다.

  2. ESC (변속기) 의 양극 (빨강) 과 음극 (검정) 을 보드에 납땜합니다.

  3. 전원 모듈의 양극 (빨강) 과 음극 (검정) 을 납땜합니다.

  4. 모터의 플러그를 ECS의 극에 맞춰에 꽂습니다.

  5. 각각의 모터를 해당하는 팔에 고정합니다.

  6. (다리의 윗부분과 나사로 결합하여) 윗판을 장착합니다.

  7. 진동 방지 폼테이프를 CUAV V5 nano 비행 컨트롤러에 부착합니다.

  8. FrSky 수신기를 양면 테이프를 이용하여 아래판에 부착합니다.

  9. 텔레메트리 모듈을 기체의 아랫판에 양면테이프를 이용하여 부착합니다.

  10. 알루미늄 GPS 지지대를 밑판에 추가한후 GPS를 부착합니다.

  11. 비행컨트롤러의 각 포트에 맞게 텔레메트리는 (TELEM1) 에, GPS 모듈은 (GPS/SAFETY) 에, RC 수신기는 (RC) 에, 모든 4 ESC’s (M1-M4) 에, 그리고 전원 모듈은(Power1) 꽂습니다.

Note 모터의 순서는 Airframe Reference > Quadrotor x에 제시되어 있습니다.

# Vehicle Configuration/Calibration

QGroundControl is used to install the PX4 autopilot and configure/tune it for the frame. Download and install QGroundControl for your platform.

Tip PX4의 설치와 설정에 대한 전체 지침은 기본 설정에서 찾을 수 있습니다.

그리고 필수 설치/설정을 수행합니다.

  • 펌웨어

  • 기체프레임

    Note Generic Quadrotor X 에어프레임을 선택합니다.(Quadrotor x > Generic Quadrotor X).

이후 다음 작업 역시 수행되어야 합니다:

  • 센서 보정
  • 지자기 센서
  • 가속도 센서
  • 수평 보정
  • 무선 설정
  • 비행 모드 > Note 이 구성을 위해 우리는 비행 모드 중 Stabilized, Altitude 그리고 Position를 리시버의 3축 스위치 (단일 채널 - 5번에 맵핑) 에 설정합니다. 이 방법이 초심자를 위한 최소 설정으로 추천됩니다.

Ideally you should also do:

  • ESC 캘리브레이션
  • 배터리
  • Safety

# 튜닝

Firmware installation sets default autopilot parameters that have been configured for the selected frame. These are good enough to fly with, but it is a good idea to tune the parameters for a specific frame build.
