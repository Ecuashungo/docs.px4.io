# 속도 센서

고정 회선 및 VTOL 프레임의 경우 속도 센서가 *을 높게 권장합니다. 자동 조종 장치에는 실속을 감지 할 수있는 다른 수단이 없으므로이 신호는 매우 중요합니다. 고정익 비행의 경우 지상 속도가 아닌 양력을 보장하는 속도입니다.

# 하드웨어 옵션

권장 디지털 속도 센서는 다음과 같습니다.

  • MEAS Spec series (e.g. MS4525DO (opens new window))

    • mRo I2C Airspeed Sensor JST-GH MS4525DO (opens new window) (mRo store)
    • Digital Differential Airspeed Sensor Kit (opens new window) (Drotek).

  • EagleTree 항공 속도 MicroSensor V3 (opens new window) (이글트리 시스템)

  • mRo 차세대 MS5525 속도 센서 (opens new window)

  • Sensirion SDP3X 차압 센서 (opens new window)

  • Holybro Digital Air Speed Sensor (opens new window)

All the above sensors are connected via the I2C bus/port.

TIP

Note Additionally, the Avionics Anonymous Air Data Computer (opens new window) can be connected to the UAVCAN bus to determine not only high-accuracy airspeed, but also true static pressure and air temperature via onboard barometer and an OAT probe.
