# Drotek XL RTK GPS

The Drotek XL RTK GPS receiver (opens new window) is a competitively priced RTK GPS module that can be configured as either base or rover. Using RTK, PX4 can get its position with centimetre-level accuracy, which is much more accurate than can be provided by a normal GPS.

# Purchase

Drotek XL RTK (opens new window) (drotek.com)

# Configuration

  • QGroundControl </ 0>을 통한 PX4의 RTK 설정 및 사용은 대부분 플러그 앤 플레이입니다 (자세한 내용은 RTK GPS </ 1> 참조).

TIP

The Drotek UCenter tool is not required for configuration - everything is handled by QGroundControl.
