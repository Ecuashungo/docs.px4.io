# 완성된 기체들

This section contains information about complete vehicles that run PX4.

Note

This is a small subset of vehicles that can run PX4. You can find others on px4.io (opens new window) and in the Airframes Reference.

  • PX4를 네이티브 버전으로 구동할 수 있는 소비자용 드론들 / "out of the box":

    • 멀티콥터
      • Teal One (opens new window)
      • NXP HoverGames KIT-HGDRONEK66 (opens new window) (hovergames.com (opens new window))
      • ModalAI m500 (opens new window)
    • VTOL
      • Vertical Technologies DeltaQuad (opens new window)

  • (다른 비행 스택을 적용했으나) PX4로 업데이트 받을 수 있는 소비자용 드론들:

    • Crazyflie 2.0
    • BetaFPV Beta75X 2S Brushless Whoop

  • (벤더가 지원하는) 커스텀 버전의 PX4를 구동하는 소비자용 드론들:

    • Multicopter
      • Yuneec Typhoon H Plus (opens new window)
      • Yuneec Mantis Q (opens new window)
      • Yuneec H520 (opens new window)
      • Airlango Mystic (opens new window)
      • Airdog II (opens new window)
      • AeroSense Aerobo (AS-MC02-P) (opens new window)
    • 고정익:
      • Sentera PXH (opens new window)
    • VTOL
      • WingtraOne Tailsitter VTOL (opens new window)
      • Flightwave Edge (opens new window)

  • Drone Development Kits

    • PX4 Vision DevKit - Multicopter used for PX4 computer vision development

  • Drone Reference Platforms

    • Intel® Aero RTF Drone (Discontinued)
