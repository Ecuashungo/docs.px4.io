# 파일 및 코드 설치

PX4 코드는 Linux 또는 Mac OS에서 개발할 수 있습니다. Ubuntu Linux LTS (opens new window) 에디션을 권장하며, 모든 PX4 대상 빌드 및 대부분의 시뮬레이터와 ROS 사용이 가능하기 때문입니다.

# 지원 대상

아래 표는 각 OS에서 구축 할 수 있는 PX 대상을 보여줍니다.

시뮬레이션: jMAVSim SITL X X X
시뮬레이션: Gazebo SITL X X
시뮬레이션: Gazebo ROS X

# 개발 환경

개발 환경 설치는 아래에서 다룹니다.

  • Mac OS
  • Linux
  • Windows

Docker에 익숙하다면 준비된 컨테이너 중 하나인 Docker Containers를 사용할 수도 있습니다.
