# 고급 기능

PX4 자동비행 프로그램의 고급 기능에 대하여 설명합니다.

  • 항공 충돌 방지: ADS-B/FLARM
  • 항공 충돌 방지: UTM
  • 컴퓨터 비전
    • 장애물 회피 (보조 컴퓨터 필요)
    • 안전 착륙 (보조 컴퓨터 필요)
    • 충돌 방지 (보조 컴퓨터 또는 비행 제어장치 센서 활용 가능)
    • 경로 계획 인터페이스 (개발자 인터페이스)
    • 모션 감지(MoCap)
    • 시각적 관성 측정 (VIO)
  • 이리듐/락블록 위성 통신 시스템
  • 정밀 착륙
  • RTK GPS
