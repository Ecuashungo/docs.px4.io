# 고급 기능

이 절에서는 PX4 비행 제어 장치의 고급 기능 관련 주제를 다룹니다:

  • 항공 충돌 방지: ADS-B/FLARM
  • 항공 충돌 방지: UTM
  • 컴퓨터 비전
    • 장애물 회피 (보조 컴퓨터 필요)
    • 안전 착륙 (보조 컴퓨터 필요)
    • 충돌 방지 (보조 컴퓨터 또는 비행 제어장치 센서 활용 가능)
    • 경로 계획 인터페이스 (개발자 인터페이스)
    • 움직임 감지(MoCap)
    • 관성 주행 시각 측정 (VIO)
  • 이리듐/락블록 위성 통신 체계
  • 정밀 착륙
  • RTK GPS
