# Jenkins CI（持续集成测试）

ci.px4.io (opens new window) 上的 Jenkins 持续集成服务器用于自动运行针对 PX4 SITL 的集成测试。

# 概述

  • 涉及的组件：Jenkins，Docker，PX4 POSIX SITL
  • 测试在 Docker Containers 内运行
  • Jenkins 执行了 2 个工作：一个用于检查每个 PR 与主控，另一个用于检查主控上的每次推送

# 测试执行

Jenkins 使用 run_container.bash (opens new window) 来启动 container，而 container 又执行 run_tests.bash (opens new window) 来编译和运行测试。

如果安装了 Docker，则可以在本地使用相同的方法：

cd <directory_where_firmware_is_cloned>
sudo WORKSPACE=$(pwd) ./Firmware/integrationtests/run_container.bash
